# Marketplace Frontend

Мультивендорный маркетплейс на Vue 3 + TypeScript.

## Технологии

- Vue 3 (Composition API)
- TypeScript
- Vite
- Vue Router
- Pinia
- Tailwind CSS
- Chart.js
- Axios

## Установка

```bash
npm install
```

## Запуск

```bash
npm run dev
```

## Сборка

```bash
npm run build
```
